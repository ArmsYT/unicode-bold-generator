# Arms | Unicode Bold Generator

This web application allows you to enter text and instantly convert it into Unicode bold text. Perfect for social media, signatures, or simply adding a unique style to your messages.

---

## 📄 Description

Cette application web vous permet de saisir un texte et de le convertir instantanément en texte gras Unicode. Idéal pour les réseaux sociaux, signatures, ou simplement pour ajouter un style unique à vos messages.

---

## ⚙️ Features

- Automatic conversion of text to Unicode bold on the fly.
- Support for French accented letters.
- Easy copying of the generated text.
- Simple and responsive interface.

---

## 🚀 Usage

Clonez le projet ou téléchargez les fichiers.
```
git clone https://github.com/ArmsYT/unicode-bold-generator.git
```

Ouvrez index.html dans votre navigateur.
Entrez le texte que vous souhaitez convertir dans le champ prévu.
Copiez le texte généré avec le bouton "Copy".
